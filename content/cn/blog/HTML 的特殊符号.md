## 1 常用的字符实体  


| 显示结果 | 描述 | 实体名称 | 实体编号 |
| --- | --- | --- | --- |
| ￠ | 分 | `&cent;` | `&#162;` |
| £ | 镑 | `&pound;` | `&#163;` |
| ¥ | 元 | `&yen;` | `&#165;` |
| § | 节 | `&sect;` | `&#167;` |
| © | 版权 | `&copy;` | `&#169;` |
| ® | 注册商标 | `&reg;` | `&#174;` |
| × | 乘号 | `&times;` | `&#215;` |
| ÷ | 除号 | `&divide;` | `&#247;` |
| < | 小于号 | `&lt;` | `&#60;` |
| > | 大于号 | `&gt;` | `&#62;` |
| & | 和号 | `&amp;` | `&#38;` |
| " | 引号 | `&quot;` | `&#34;` |
| ' | 撇号 | `&apos;` (IE不支持) | `&#39;` |
|  | 空格（推荐使用） | `&nbsp;` | `&#160;` |
|  | 半角空格 | `&ensp;` |  |
|  | 全角空格 | `&emsp;` |  |
